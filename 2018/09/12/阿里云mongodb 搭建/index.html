第二步：安装mongodb

关闭数据库

mongo
use admin
db.shutdownServer()

删除mongodb

sudo yum erase $(rpm -qa | grep mongodb-org)
sudo rm -r /var/log/mongodb（日志）
sudo rm -r /var/lib/mongo（数据）

停止mongod服务

sudo service mongod stop
